# ChakshuFramework
An amazing framework
# Browser Stack #
## PlayWright POM Framework ##
### Features ###
- Page Object Model (POM) design pattern for better code organization and maintainability.
- **Pytest-based tests** with parameterization for multiple data set
- **Excel-driven test data** using `openpyxl`
- **Logging** for test execution into `logs/framework.log`
- **Screenshot folder** on test failures and validation saved in `screenshots/`
- **Report folder** for storing test report
- **Configurable and extensible structure** for easy scaling
